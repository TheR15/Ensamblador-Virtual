Gabinete

Descripción

El gabinete, también conocido como caja o chasis, es la estructura que alberga y protege todos los componentes internos de una computadora. Proporciona soporte físico, protección contra el polvo y los elementos, y ayuda en la gestión térmica del sistema.

Función

Las principales funciones del gabinete incluyen:

  • Proteger los componentes internos de daños físicos y contaminantes ambientales.
  • Proporcionar puntos de montaje para la placa madre, fuente de alimentación, unidades de almacenamiento, etc.
  • Facilitar el flujo de aire para la refrigeración de los componentes.
  • Organizar y gestionar el cableado interno.
  • Proporcionar puertos de E/S frontales para fácil acceso.

Características Principales

  • Tamaño y factor de forma: ATX, Micro-ATX, Mini-ITX, etc.
  • Bahías de expansión: Para unidades ópticas, discos duros, SSDs.
  • Puertos frontales: USB, audio, botones de encendido y reinicio.
  • Gestión de cables: Orificios y canales para organizar el cableado.
  • Ventilación: Aberturas y puntos de montaje para ventiladores.
  • Materiales: Generalmente acero, aluminio o plástico.
