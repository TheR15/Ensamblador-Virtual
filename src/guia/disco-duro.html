Disco Duro

Descripción

El disco duro es un dispositivo de almacenamiento no volátil que se utiliza en las computadoras para guardar y recuperar información digital. A diferencia de la memoria RAM, el disco duro mantiene los datos incluso cuando la computadora está apagada.

Función

Las principales funciones del disco duro incluyen:

  • Almacenar permanentemente el sistema operativo, programas y archivos del usuario.
  • Proporcionar acceso a los datos almacenados cuando sea necesario.
  • Mantener una estructura organizada de archivos y directorios.
  • Servir como memoria a largo plazo de la computadora.

Tipos y Características

  • HDD (Hard Disk Drive): Utiliza discos magnéticos giratorios para almacenar datos.
  • SSD (Solid State Drive): Utiliza memoria flash para un acceso más rápido a los datos.
  • Capacidad: Medida en gigabytes (GB) o terabytes (TB).
  • Velocidad de rotación (para HDD): Medida en RPM (revoluciones por minuto).
  • Velocidad de transferencia: Cantidad de datos que pueden leerse o escribirse por segundo.
  • Interfaz: SATA, NVMe, etc., que determinan cómo se conecta el disco a la placa madre.
