Placa Madre

Descripción

La placa madre, también conocida como tarjeta madre o motherboard, es el componente principal de una computadora. Actúa como la columna vertebral del sistema, conectando y permitiendo la comunicación entre todos los demás componentes del ordenador.

Función

Las principales funciones de la placa madre incluyen:

  • Proporcionar conexiones para los componentes del sistema.
  • Distribuir la energía eléctrica a los componentes conectados.
  • Contener el BIOS (Sistema Básico de Entrada/Salida).
  • Alojar el chipset que gestiona el flujo de datos entre el procesador, la memoria y los periféricos.

Componentes Principales

  • Socket de CPU: Donde se instala el procesador.
  • Ranuras de memoria RAM: Para instalar los módulos de memoria.
  • Chipset: Controla la comunicación entre los componentes.
  • Ranuras de expansión (PCI, PCIe): Para tarjetas adicionales como GPU.
  • Conectores SATA: Para conectar discos duros y unidades ópticas.
  • Puertos de E/S: USB, audio, red, etc.
